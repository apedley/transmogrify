# Transmogrify

Convert files by renaming them.
**Transmogrify** watches your folders and converts files instantly, right from Finder.

## Install instructions

Installation

1.	Download the latest release from the Releases page.
2.	Open the .dmg file.
3.	Drag **Transmogrify** into your Applications folder.
4.	Launch **Transmogrify** from Applications.

## About this build

This repository provides a direct-download build of Transmogrify for people who prefer installing apps outside the Mac App Store. A Mac App Store version is also available for users who want App Store distribution and system-managed updates.
